Website | Simone and Peter's Airbnb

Source code The website

Project Background

This project involved creating a website for a fictitious business, Simone and Peter's Airbnb.

Discovery

Printed photos and details about the Airbnb business, together with project requirements were supplied and used to design and develop the website.

Design

Good design principles were considered and implemented for visual elements, navigation and accessibility.

Development and Testing

The website was developed for desktop devices with breakpoints for mobile users. It was written in HTML5 and CSS programming languages and includes an embedded YouTube clip, a gif images slideshow, a Google map and links to local tourism sites.

The accessibility of the website was tested and evaluated against WebAIM's Guidelines and with their WAVE tool to ensure accessibility guidelines were followed.

Deployment

Deployment of the website to a hosting provider was not required for this project. However, the website was added to GitHub Pages after this project.

Evaluation

Ongoing support and maintenance was not needed. However, there was a requirement for the client to modify images, font styles and links for the website in future. A training manual with screenshots were created to support them with this.
